# Analytics Metrics

CloudLabs AI Proctor tracks usage and response quality metrics to evaluate how effectively the assistant supports learners.

## Core Metrics

- Total chat sessions
- Total user messages
- Total AI responses
- Most common issue categories
- Knowledge base source usage
- Average confidence score
- Helpful vs Not Helpful feedback
- Escalation count

## MVP Storage

For the MVP, analytics are stored in lightweight JSON files:

- `analytics/events.json`
- `analytics/feedback.json`

## Future Metrics

Future versions may include:

- Average resolution time
- Learner completion rate
- Lab abandonment rate
- Agent accuracy score
- Human escalation reduction
