# cloudlabs-ai-proctor

Brainy (project codename: cloudlabs-ai-proctor) is an AI-powered virtual lab copilot embedded inside CloudLabs hands-on training environments. It sits alongside the learner's lab guide and VM, and helps them:

•	Troubleshoot common lab issues (VM not loading, login failures, Azure Portal errors, etc.)
•	Get step-by-step guidance for the exercise/task/step they're currently on
•	Ask general conceptual questions about Azure, Microsoft Fabric, Power Platform, GitHub Copilot, and other workshop topics
•	Get pointed to a human (CloudLabs Support) when the AI genuinely can't help

Instead of a learner waiting for a human proctor to respond in a live chat, Brainy gives an immediate, context-aware answer generated from CloudLabs' own knowledge base plus 

Azure OpenAI reasoning — with a safety net of general web search and, ultimately, a real support email when nothing else applies.

Live product name in the UI: "Brainy — AI Lab Copilot"
