# Access Denied

## Issue
The attendee receives an access denied or permission error.

## Possible causes
- Wrong account is being used
- Required role is missing
- Tenant setting is disabled
- License or trial is not active

## Resolution steps
1. Confirm the user is signed in with the correct lab account.
2. Ask the user to sign out from other accounts.
3. Open the portal in InPrivate/Incognito mode.
4. Check whether the required role or license is available.
5. Escalate if admin permission is needed.

## Escalation required
Yes, if tenant admin or role assignment is required.
