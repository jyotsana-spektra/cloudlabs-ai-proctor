# VM Not Loading

## Issue
The attendee cannot open or access the lab VM.

## Possible causes
- VM is still starting
- Browser issue
- Network issue
- CloudLabs session issue

## Resolution steps
1. Wait 2–3 minutes.
2. Refresh the CloudLabs page.
3. Try opening the VM again.
4. Use a supported browser such as Microsoft Edge or Chrome.
5. If the issue continues, escalate to a human proctor.

## Escalation required
Yes, if the VM does not load after retrying.
